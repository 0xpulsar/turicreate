Detailed Description

Defines a memory mapped Sframe. i.e. an SFrame mapped into a single pointer in memory via the pagefault handler.

Only FLOAT, INTEGER, VECTOR columns are supported. For Integer columns , missing values are mapped to 0. For float columns , missing values are mapped to NaN.
