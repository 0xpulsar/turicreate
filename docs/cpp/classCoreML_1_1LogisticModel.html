Detailed Description

Reader/Writer interface for a GLM.

A construction class that, in the end, outputs a properly constructed specification that is gauranteed to load in an LinearModelSpec class.

Enforces type invariant conditions.

Enforces type invariant conditions for this TransformSpec, given a list of allowed feature types and a proposed feature type. If the proposed feature type is in the list of allowed feature types, this function returns a Result with type ResultType::NO_ERROR. If the proposed feature type is not in the list of allowed feature types, this function returns a Result with type ResultType::FEATURE_TYPE_INVARIANT_VIOLATION.

Parameters
allowedFeatureTypesthe list of allowed feature types for this transform.
featureTypethe proposed feature type of a feature for this transform.
Returns
a Result corresponding to whether featureType is contained in allowedFeatureTypes.
